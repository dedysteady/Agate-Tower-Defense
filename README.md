# Additional Features
- Backsound theme and Sound Fx
- Simple Wave System (3 wave)
- Enemy x2 every next wave

# How to Play
- Download the Repository
- Extract build.zip
- Open the Folder Build
- Then Run .exe
- Setup prefered resolution size
- ENJOY !!!
